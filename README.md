# NyarukoLogin 2

一个通用的用户登录系统，目标是本身作为一个服务，为外部业务系统进行服务，可多开，可负载均衡。

- 注意：这个程序尚未做完，请勿使用。
- 要查看旧版本代码和此处的说明，请前往 v2016_expired 分支。

## 开发进度
